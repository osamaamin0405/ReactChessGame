# ReactChessGame
React Type Script Chess Multi Player With AI Game
## Live Preview
  - https://chess-eng.netlify.app/ (Desktop only)

## How you can install it

  - 1
    `shell git clone  https://github.com/osamaradwan2003/ReactChessGame.git`
  - 2
    `shell  npm run dev `

## Requirements
  - Node.js
  - npm package manager
